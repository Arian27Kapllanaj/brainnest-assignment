1. Minecraft
    • Minecraft is an open world sandbox game where you can build whatever you desire. It was created by a company called Mojang in the Java programming language.
    • This game encourages creativity, as you need to build structures all on different blocks you have
    • It can also teach some fundamental programming with an aspect of the game called redstone
  2. Geometry Dash
    • Geometry Dash is a music beat based game where you play as a cube and jump over spikes.
    • It is an incredibly challenging game, and difficult that ranges from easy to demon
    • You can also download other people's levels, so the content is seemingly endless.
